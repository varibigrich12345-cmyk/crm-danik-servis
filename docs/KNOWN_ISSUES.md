# Известные проблемы

## Критические
1. **Ошибка "Не удалось определить пользователя"**
   - Причина: profile не загружается при создании заявки
   - Файл: src/components/claims/ClaimFormDialog.tsx
   - Статус: в процессе исправления

2. **Браузер виснет после входа**
   - Причина: возможен бесконечный цикл в useAuth.ts
   - Статус: требует диагностики

## Решённые
- RLS блокировал доступ к profiles → отключен
- 406 ошибка при запросе профиля → заменили .single() на .maybeSingle()
- Редирект после входа не работал → добавили setTimeout
