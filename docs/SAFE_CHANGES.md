# Безопасные изменения

## ✅ Можно менять безопасно
- Стили (Tailwind классы)
- Тексты и переводы
- Добавление новых страниц
- Добавление новых компонентов в components/

## ⚠️ Менять осторожно (делать бэкап)
- useAuth.ts - хук авторизации
- App.tsx - роутинг
- ClaimFormDialog.tsx - форма заявки

## ❌ Не трогать без понимания
- supabase.ts - клиент БД
- Supabase migrations
- .env файлы

## Перед изменением критичных файлов
1. git add . && git commit -m "backup before changes"
2. Скопировать файл в docs/backups/
3. Внести изменения
4. Протестировать
5. Если сломалось: git checkout -- filename
