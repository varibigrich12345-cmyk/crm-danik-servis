# SQL Скрипты для базы данных

## Применение скриптов

### Через Supabase Dashboard
1. Откройте [Supabase Dashboard](https://supabase.com/dashboard)
2. Выберите ваш проект
3. Перейдите в **SQL Editor**
4. Скопируйте содержимое нужного SQL файла
5. Вставьте в редактор и нажмите **Run**

### Через psql (командная строка)
```bash
psql -h db.mchhjqmgdhtzrpcevgdp.supabase.co -U postgres -d postgres -f works_dictionary.sql
```

## Доступные скрипты

### works_dictionary.sql
Создает таблицу справочника работ и заполняет её полным списком работ автосервиса (более 150 позиций).

**Категории работ:**
- Диагностика
- Ходовая часть
- Тормозная система
- Двигатель
- Система охлаждения
- Трансмиссия
- Рулевое управление
- Электрика
- Кондиционер
- Выхлопная система
- Кузов и стекла
- Шиномонтаж

**Применение:**
```sql
-- Скрипт автоматически создаст таблицу и заполнит данными
-- ON CONFLICT (name) DO NOTHING - предотвращает дубликаты при повторном запуске
```

### parts_manufacturers.sql
Создает таблицу справочника производителей запчастей и заполняет её списком производителей (более 120 брендов).

**Категории производителей:**
- OEM (оригинальные производители)
- Подвеска
- Тормоза
- Фильтры
- ГРМ (ремни и ролики)
- Сцепление
- Охлаждение
- Зажигание
- Электрика
- Масла
- Приводы
- Подшипники
- Бюджетные бренды

**Применение:**
```sql
-- Скрипт автоматически создаст таблицу и заполнит данными
-- ON CONFLICT (name) DO NOTHING - предотвращает дубликаты при повторном запуске
```

## Важно
- Все скрипты отключают RLS (Row Level Security) для упрощения разработки
- В продакшене рекомендуется включить RLS и настроить политики безопасности
